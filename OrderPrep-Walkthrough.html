5Ô∏è‚É£Kitchen Planning & Menu Management

Two Ways to Plan Your Menu

üí° Which Mode to Choose?
Slot Mode is recommended if you have 20+ menu items. It shows only your best items for the day.
Full Menu Mode works better for smaller menus (under 15 items).

Option 1: Slot Mode (Recommended for Large Menus)

Show only 8-15 curated slots - perfect for businesses with many items:

  1. Go to Kitchen tab
  2. Toggle to "Slot Mode" (at the top)
  3. Choose how many slots: 8, 10, 12, or 15 items
  4. For each slot:
    • Click dropdown and select which menu item
    • Enter quantity you plan to cook
    • Price auto-updates from the menu
  5. Click "Publish Today's Menu"
  6. Only these curated items appear to customers

‚ú® Why Slot Mode is Great for Large Menus

‚Ä¢ Shows customers only your best/popular items

‚Ä¢ Reduces decision paralysis (easier for customers to choose)

‚Ä¢ Helps you plan exactly what to cook each day

‚Ä¢ Can rotate different items daily to keep menu fresh

‚Ä¢ Reduces waste by limiting your offerings

Option 2: Full Menu Mode (For Small Menus)

Show ALL available items and their quantities:

  1. Go to Kitchen tab
  2. Make sure "Full Menu Mode" is selected
  3. Edit quantities for each menu item:
    • Enter how many you plan to cook
    • Price auto-updates from menu
  4. Click "Publish Today's Menu"
  5. All items appear in Orders tab and WhatsApp menu

Adding New Menu Items

  1. Click "+ Add Menu Item" button
  2. Fill in form:
    • Name: Item name (e.g., "Honey Pork Ribs")
    • Description: What's in it (e.g., "Slow cooked in honey soy sauce")
    • Category: Main, Dessert, Snack, Beverage
    • Price: Base selling price
  3. Click "Add Item"

Editing Menu Items

  1. Go to "Menu List" tab
  2. Click "Edit" button on item
  3. Modify any field (name, price, description, category)
  4. Click "Save Changes"

ü§ñ AI Menu Recommendations

After taking orders, Kitchen tab shows:

‚úì COOK MORE: Popular items running low

‚úì COOK LESS: Slow-moving items with leftovers

‚úì MAINTAIN: Items with perfect demand

These are based on 7-day sales history!

üí° Pro Tip

When publishing menu, check which items are running low and which have too much stock. Adjust quantities based on recommendations to reduce waste and sold-outs.

  4. This is your data insurance - KEEP IT SAFE!
‚ö†Ô∏è CRITICAL: Your data lives ONLY in browser. If you clear cache, close browser tab wrong way, or format computer - ALL DATA GONE. Download backup every week!

  1. Click "Upload Backup" button
  2. Select your previously downloaded JSON file
  3. Confirm - this REPLACES all current data
  4. Data restores completely
üí° Backup Strategy: Download backup every Friday evening. Keep 4 weekly backups. If something goes wrong, you can restore from last week's backup.

9Ô∏è‚É£Tips & Best Practices

Daily Workflow

üåÖ Morning (Before Orders Start)

  1. Check Dashboard for yesterday's unpaid amounts
  2. Send payment reminders if needed
  3. Review Kitchen recommendations
  4. Decide what to cook based on analytics
  5. Publish today's menu

‚òÄÔ∏è During Day (Taking Orders)

  1. Add each order in Orders tab immediately
  2. Mark as "Paid" if customer pays immediately
  3. Mark as "Pending" if paying later
  4. Use Flash Sale for items needing to clear

üåÜ Evening (End of Day)

  1. Review today's revenue on Dashboard
  2. Check stock levels in Prep
  3. Note which items sold well/didn't sell
  4. Send payment reminders to pending customers

Maximizing Flash Sales

  • Use only for items that MUST move (expiring, old stock)
  • Check "Flash Sale Impact" to see real cost of discounts
  • If spending 300+ AED/day on discounts, cook less tomorrow
  • Track which items repeat in flash sales - reduce cooking quantity

Improving Payment Collection

  • Send Friendly Reminder after 1 day
  • Send Professional Follow-up after 3 days
  • Send Final Notice after 5+ days
  • Reduce credit to "Risk" badge customers
  • Offer small discount (1-2 AED) for immediate payment

Smart Menu Planning

  • Follow "COOK MORE" recommendations for popular items
  • Cut quantities of "COOK LESS" items by 20-30%
  • This reduces waste AND sold-outs
  • Saves time and ingredients

Using Analytics

  • Track customer preferences over time
  • Identify seasonal trends
  • See which customers are most valuable
  • Understand your revenue patterns
