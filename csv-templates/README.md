# CSV Import Templates for OrderPrep

This folder contains CSV templates for importing menu items and customer data into OrderPrep.

## üìã Menu Items Template (`menu-template.csv`)

### Required Fields:
- **id**: Unique identifier (e.g., "1", "2", "menu_001")
- **name**: Menu item name (e.g., "Honey Pork Ribs")
- **price**: Price in AED (number only, e.g., 15)
- **category**: Category (Main, Dessert, Snack, Beverage)
- **isAvailable**: true or false

### Optional Fields:
- **description**: Item description

### Example:
```csv
id,name,price,category,description,isAvailable
1,Honey Pork Ribs,15,Main,Slow cooked pork ribs in honey soy sauce,true
2,Brownies,10,Dessert,Chocolate brownies,true
```

### Notes:
- Do NOT include daily limits (dailyLimit) - these are set in Kitchen Planning
- Categories: Main, Dessert, Snack, Beverage
- Keep descriptions concise (under 100 characters)

## üë• Customers Template (`customers-template.csv`)

### Required Fields:
- **id**: Unique identifier (e.g., "c1", "cust_001")
- **name**: Customer name
- **phone**: UAE phone number (format: 971501234567)
- **totalOrders**: Total number of orders (default: 0)
- **totalSpent**: Total amount spent in AED (default: 0)

### Optional Fields:
- **unitNumber**: Unit/apartment number (e.g., "501")
- **floor**: Floor number (e.g., "5")
- **building**: Building name (e.g., "A", "Tower A")
- **location**: Additional location info (e.g., "Control Tower", "Reception")
- **notes**: Customer notes/preferences

### Example:
```csv
id,name,phone,unitNumber,floor,building,location,notes,totalOrders,totalSpent
c1,Sarah,971501234567,501,5,A,Control Tower,VIP customer,12,540
c2,John,971559876543,803,8,A,Control Tower,,3,135
```

### Phone Number Format:
- Include country code: 971 (UAE)
- No spaces or dashes
- Example: 971501234567 (not +971 50 123 4567)

### Notes:
- Leave optional fields empty if not available (don't use "N/A")
- totalOrders and totalSpent will update automatically as orders are placed
- Use consistent building names across customers for autocomplete

## üîß How to Import

### In OrderPrep App:

1. **Menu Items:**
   - Go to **Settings** tab
   - Scroll to **Import Data** section
   - Click **Choose File** under "Import Menu Items (CSV)"
   - Select your CSV file
   - Choose import option:
     - **Replace All**: Removes existing menu and imports CSV
     - **Merge**: Adds new items, updates existing by ID

2. **Customers:**
   - Go to **Settings** tab
   - Scroll to **Import Data** section
   - Click **Choose File** under "Import Customers (CSV)"
   - Select your CSV file
   - Choose import option:
     - **Replace All**: Removes existing customers and imports CSV
     - **Merge**: Adds new customers, updates existing by ID
